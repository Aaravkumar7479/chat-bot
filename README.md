# Chatbot

## Overview
A simple AI-powered chatbot application that allows users to interact through a conversational interface. The chatbot processes user input and generates intelligent responses, making it suitable for learning conversational AI concepts and building interactive applications.

## Features
- Interactive chat interface
- Real-time conversation
- User-friendly design
- Input validation
- Easy to extend with new features
- Clean and modular code structure

## Tech Stack
- Java
- Object-Oriented Programming (OOP)
- Collections Framework
- File Handling (optional)
- Git & GitHub

## How to Use
1. Clone this repository.
2. Open the project in your preferred Java IDE.
3. Build and run the application.
4. Start chatting with the bot through the console or interface.

## Run in GitHub Codespaces
1. Click the **Code** button on GitHub.
2. Select the **Codespaces** tab.
3. Create a new Codespace.
4. Wait for the environment to load.
5. Compile the project:
   ```bash
   javac Main.java
   ```
6. Run the application:
   ```bash
   java Main
   ```
